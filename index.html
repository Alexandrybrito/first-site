Bem-vindo ao meu primeiro site!

Este site é uma atividade do módulo 3 do curso fullstack jr, da +praTi, etapa front-end, posposto para balizar os conhecimentos adiquiridos em aulas. O desafio é desenvolver um site front-end, que atenda os requisitos solicitados (como por exemplo, estruturaçao do html e responsívidade em css). Ele tem como objetivo colocar em prática os elementos de HTML e CSS estudados nesse módulo.


Curiosidade: Você sabia que o html e o css são responsáveis por praticamento todo front-end de sites, páginas e outras aplicações na internet?

HTML5

HTML é a sigla para Linguagem de Marcação de Hipertexto, e pode ser definida como uma linguagem de computador, mas não de programação. Ela é amplamente utilizada para desenvolver páginas da web, uma vez que documentos HTML são facilmente interpretados pela maioria dos navegadores.

CSS3

CSS é a sigla para Cascading Style Sheet e sua função principal é atribuir estilo a elementos escritos em HTML, ou seja, o CSS cuida da parte visual e estética de um site: cores, fontes, espaçamentos entre parágrafos, tabelas, variações de layouts, ajustes de imagens, tudo isso entra como estilização.
